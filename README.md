# Workout Tracker

A single-page workout, nutrition, and body-weight tracker. Open `index.html` in any browser.

- **Workouts** — grouped exercise tables (back/bicep, chest/tricep, legs, cardio) with per-exercise
  progress charts, set logging, and form-demo links.
- **Nutrition** — natural-language food logging with automatic macros (via the Nutritionix API).
- **Body** — daily weight tracking with a chart, plus Apple Health `export.xml` import.

Data syncs across devices through a private cloud store and is cached locally for offline use.

This is a static site — no build step, no server. It can be hosted as-is on GitHub Pages.
